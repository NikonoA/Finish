Git

  • git status - проверка изменений в репозитории
  • git commit - создание коммита
  • git log - выведение списка всех коммитов
  • git push - перенос изменений на удалённый репозиторий
  • git pull - перенос изменений на локальный репозиторий
  • git add - добавление файла
